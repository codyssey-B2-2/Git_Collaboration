# Git_Collaboration

3인 팀이 GitHub Flow, Issue, PR, Review, Conflict, Troubleshooting 흐름을 실습하는 저장소입니다.

## 팀원 소개

- [VectorSophie](team/VectorSophie.md)

기본 Github Flow 규칙
- main은 항상 배포 가능 브랜치로 유지합니다.
- 모든 작업은 feature/<name>/<topic> 브랜치에서 진행합니다.
- PR과 리뷰를 거쳐 병합하면서 작업 이유와 검증 기록이 남깁니다.
